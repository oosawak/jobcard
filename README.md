# jobcard

スマートフォン向けの仕事カード UI をまとめたデモです。

## 構成

- `src/` - 実装ソース
- `docs/` - GitHub Pages 公開用

## 使い方

GitHub Pages の公開先を `main` ブランチの `docs/` に設定してください。

設定場所:

1. GitHub リポジトリの `Settings`
2. `Pages`
3. `Build and deployment`
4. `Source` を `Deploy from a branch`
5. `Branch` を `main`
6. `Folder` を `/docs`
